# Página Web Pessoal — Avaliação Final

Este projeto consiste no desenvolvimento de uma página web pessoal, utilizando HTML, CSS e JavaScript, conforme solicitado pelo professor.

A página foi desenvolvida com foco em simplicidade, organização do código e uso correto das tecnologias básicas do desenvolvimento web.

---

# Tecnologias Utilizadas

- **HTML5** → Estrutura da página  
- **CSS3** → Estilização e layout  
- **JavaScript** → Interatividade e validação de formulário  

---

# Estrutura do Projeto

- index.html   # Página principal
- style.css    # Estilos da página
- script.js    # Funcionalidades em JavaScript
- README.md    # Documentação do projeto
